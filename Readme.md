1.What is the difference between var, let, and const?
Ans: In var we can Reassign and Redeclare . In var no block scope. 
In let we can Reassign but we can not Redeclare . It has a block scope. 
In const We can not Reassign and Redeclare . It has a block scope.

3.Difference between map(), filter(), and forEach()
Ans: map() transform every element in new array but main array not changes 
filter() transform element by condition in new array 
forEach() traversing every element in main array
